# EDEN SKYE ADMIN Bridge Manifest

## Source Manifest

Machine-readable bridge registry: `config/eden-skye-admin-bridge-registry.json`.

## Bridge Pattern

The uploaded black chat/admin UI package establishes the operating pattern:

1. Agent proposes a task.
2. Preflight checks validate scope and available systems.
3. Policy check detects protected actions.
4. Approval request is created when needed.
5. Safe draft command is queued.
6. Sandbox or preview execution runs when available.
7. Receipt and evidence are written.
8. Governance decides whether the next gate is available.

## Live Mutation Rule

All bridges default to draft-only operation. Production mutation requires an approval receipt tied to a specific protected action.
