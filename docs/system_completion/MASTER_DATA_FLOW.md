# MASTER_DATA_FLOW

## Purpose
Defines Eden Skye Studios data movement from discovery through revenue and learning.

## Flow
Research signals -> scored ideas -> scripts -> media jobs -> approvals -> Metricool drafts -> publishing gates -> analytics -> learning memory -> repurpose queue -> monetization routes.

## Required Controls
Every state change must include source, timestamp, workflow key, approval status, dry_run flag, and audit log reference.
