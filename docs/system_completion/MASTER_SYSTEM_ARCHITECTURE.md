# MASTER_SYSTEM_ARCHITECTURE

## Purpose
Canonical architecture for Eden Skye Studios OS.

## Operating Stack
GPT/AUTO_BUILDER = brain and governance.
n8n = workflow harness.
Supabase = memory, queues, audit, approvals.
Vercel = runtime APIs, crons, triggers.
GitHub = source artifacts.
Drive = canon and asset archive.
Metricool = social drafts and analytics.
Shopify/Stripe = commerce and payments.
Canva/Adobe/HeyGen/Runway/Xyla/Higgsfield = creative execution.

## Core Loop
Research -> Score -> Script -> Prompt -> Gate -> Generate -> QA -> Draft Schedule -> Publish Gate -> Analytics -> Learn -> Repurpose -> Monetize.

## Default Mode
Dry-run, draft-only, human approval required.
