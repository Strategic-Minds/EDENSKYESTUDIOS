# Eden's Closet Page Prompt Pack

Use these prompts when generating code or recreating route layouts. Each prompt maps to one standalone page image. Do not combine pages into a collage.

## Global Generator Instructions

You are building a Next.js App Router PWA for Eden's Closet. Use the supplied page image for one route only. Recreate the layout as real React/HTML/CSS. Do not embed the screenshot as the UI. Do not crop from a board. Do not use page screenshots as model assets. Use approved standalone model/environment assets from the manifest only.

## 01 Home `/closet-v2`

Create a black neon Eden's Closet member home page. First viewport has ES nav, large Eden's Closet title, exclusive access copy, feature checklist, CTA to enter closet, and a neon closet hero image. Below it, include a four-step how-it-works row and six feature cards.

## 02 Select Model `/closet-v2/models`

Create a model selector grid with 10 model cards, dark cards, hot pink active state, model image, city, and profile CTA. Include a Black Card membership strip at bottom with quick access icons.

## 03 Model Profile `/closet-v2/models/alexis-voss`

## 05 Environments `/closet-v2/environments`

## 06 Full Body 360 Viewer `/closet-v2/viewer`

## 07 AI Chat `/closet-v2/chat`

## 09 Dashboard `/closet-v2/dashboard`
