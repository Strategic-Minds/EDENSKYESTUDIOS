# Workflow Map

## Purpose
Map Eden Skye Studios from source truth to execution.

## Chain
AUTO_BUILDER -> GPT Workbook Library -> Drive Canon -> GitHub -> Vercel Workflows -> Supabase Memory -> Connectors -> Human Gate Dashboard.

## Operating Rule
Every workflow starts in sandbox/draft mode and escalates to a human gate before public publishing, store mutation, schema change, deployment, paid ads, persona change, or governance/source-truth mutation.

## Required Runtime Workflows
- discover-content
- plan-content
- create-content
- validate-content
- schedule-content
- analyze-content
- optimize-content
- admin-digest
- approval-gate
- failure-recovery
- connector-sync
- monthly-audit

## Required Loop
Discover -> Plan -> Create -> Validate -> Gate -> Schedule -> Publish only if approved -> Analyze -> Learn -> Optimize -> Audit.
