# EDEN SKYE ADMIN System Brief

## Purpose

EDEN SKYE ADMIN is the black luxury backend command center for Eden Skye Studios and AUTO BUILDER operations. It is the operational dashboard for planning, building, validating, approving, previewing, generating media, drafting distribution, collecting evidence, analyzing results, optimizing systems, and scaling approved workflows.

## Design Contract

- First screen is the actual admin operating dashboard.
- Black background, white typography, hot pink/neon Eden Skye accents.
- Dense backend command-center layout.
- No beige UI.
- No generic SaaS homepage.
- The uploaded black chat/admin UI package is the style and workflow source reference.

## Operating Loop

PLAN -> BUILD -> VALIDATE -> APPROVE -> DEPLOY PREVIEW -> OPERATE -> GENERATE MEDIA -> CREATE SOCIAL DRAFTS -> ANALYZE -> OPTIMIZE -> SCALE

## Autonomy Boundary

The system enables maximum safe autonomy for inspection, planning, drafting, queueing, validation, evidence collection, receipts, preview preparation, media packets, and optimization recommendations.

Protected live actions remain blocked unless Jeremy explicitly approves them through the approval gate.
