# EDEN SKYE ADMIN Connector Map

## GitHub

Use for repo inspection, branch state, PR state, issues, Actions, workflow artifacts, and branch-safe writes.

## Vercel

Use for preview deployments, build logs, deployment state, and browser-accessible preview URLs. Production promotion is approval-gated.

## Supabase

Use for registry reads, bridge queue, receipts, approval requests, and SQL validation checks. Production schema mutation is approval-gated.

## Google Drive

Use as source truth for assets, manifests, operating docs, page boards, image source manifests, quarantine, and release evidence.

## Gmail

Use for inbox triage, action queues, and draft replies. Sending to real customers is approval-gated.

## Google Calendar

Use for availability lookup and scheduling briefs. Creating events with external attendees is approval-gated.

## Shopify

Use for product/storefront planning, draft packets, and bridge readiness. Live product, payment, theme, or discount changes are approval-gated.

## HeyGen And Media Generation

Use for video/avatar packets and generated media pending review. Public approval state must be explicit before public use.

## Social Automation

Use for content plans, drafts, scheduling packets, analytics review, and winner cloning. Public posting is approval-gated.

## Browser / Playwright

Use for screenshots, UI evidence, funnel checks, and regression proof. It should not perform purchases, publishing, or credentialed mutation without approval.
