# EDEN SKYE ADMIN Operator Playbook

## Daily Command-Center Flow

1. Open `/admin`.
2. Review readiness score, active blockers, bridge status, and next best autonomous action.
3. Open Agent Console and review task queue, failed jobs, and receipts.
4. Open Git / Vercel Ops and confirm Draft PR, preview, test, build, and visual evidence state.
5. Open Drive OS and confirm source truth, missing assets, quarantine, and normalization queue.
6. Open Media Factory and review generated_pending_review assets.
7. Open Social Automation and approve or reject drafts before public posting.
8. Open Approval Gates before any protected action.
9. Open Evidence Center before moving a PR, promoting preview, or approving visuals.

## Safe Autonomous Actions

- Inspect connected systems
- Generate plans and builder docs
- Create implementation packets
- Queue draft commands
- Generate media prompts and packets
- Draft social content
- Draft Gmail replies
- Produce calendar briefs
- Collect screenshots and logs
- Write receipts
- Recommend optimization steps

## Human Approval Required

Human approval is required for production deploy, live commerce mutation, Supabase production schema changes, public social publishing, sending customer email, external calendar events, domain/billing changes, secret rotation, destructive Git operations, PR movement, merge, and release.

## Recovery

If a job fails, keep evidence, mark the receipt failed, route it to recovery, and do not retry protected actions without a fresh approval.
