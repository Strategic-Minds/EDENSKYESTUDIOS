# SHOPIFY_MONETIZATION_STANDARD

Shopify is the commerce presentation layer.

Allowed Draft Artifacts:
- product drafts
- collection drafts
- landing page drafts
- bundle concepts

Forbidden Without Approval:
- product activation
- pricing changes
- checkout changes
- payment configuration changes

Revenue Ladder:
1. digital starter products
2. audits/services
3. subscriptions
4. licensing
5. sponsorships

Gate:
Commerce writes require explicit approval.
