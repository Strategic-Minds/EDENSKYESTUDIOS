# Eden Skye Shopify Theme Branch

This branch contains a native Shopify Online Store theme structure at the repository root so Shopify can connect to the branch directly through GitHub.

Required theme folders at root:

- `layout/`
- `templates/`
- `sections/`
- `snippets/`
- `assets/`
- `config/`
- `locales/`

Use this branch for Shopify theme editing and GitHub connection. The Next.js/Vercel storefront remains a separate headless app surface and should not be uploaded to Shopify as a theme.

Launch safety:

- No live Shopify product creation is included.
- No payment activation is included.
- Product, payment, and publishing actions remain merchant-approved.
- Upload approved images through Shopify theme editor image pickers or Shopify Files.
