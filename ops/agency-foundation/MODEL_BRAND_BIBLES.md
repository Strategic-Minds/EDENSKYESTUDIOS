# Eden Skye Studios Model Brand Bibles

## Universal Model Rules

All Eden Skye Studios models are fictional adult AI avatars. They must be presented as fictional/digital models, never as real humans.

Universal visual quality:

- realistic but polished
- strong face continuity
- natural anatomy
- premium lighting
- no text/logos/watermarks in source images
- no age ambiguity
- no explicit sexual content
- no real-person imitation

Universal production requirements:

- each model has a stable ID
- each image/video job references source image IDs
- each generated asset receives a receipt
- public or client-facing use requires approval
