# Eden Skye Studios Source Of Truth

## Objective

Eden Skye Studios is the governed brand, media, content, commerce, and automation operating system for Eden Skye: a fictional adult AI luxury creator brand.

The system exists to create a controlled autonomous pipeline for:

- brand-safe image generation
- avatar and video production
- content drafting and repurposing
- social draft queues
- Shopify/store launch support
- Supabase-backed telemetry and state
- Vercel/Next.js control surfaces
- Drive/GitHub source management
- Auto Builder orchestration and validation

## Operating Doctrine

Drive stores media assets and working artifacts.
GitHub stores source truth, manifests, code, contracts, prompts, and receipts.
Supabase stores state, telemetry, queues, approvals, and operational logs.
Vercel runs frontend, APIs, and cron triggers.
Auto Builder orchestrates packets, validation, promotion, and controlled improvement.
HeyGen, Runway, Descript, Canva, Metricool, Shopify, and other tools are execution layers, not governance layers.
