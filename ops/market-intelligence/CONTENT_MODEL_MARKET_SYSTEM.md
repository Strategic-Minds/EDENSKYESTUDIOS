## How The 56-Image System Happens

1. Approve one model archetype and lock her identity brief.
2. Generate the 8 HeyGen source images first.
3. Run face-continuity review and reject identity drift.
4. Generate expression, full-body, wardrobe, environment, commerce, and social/video source sets.
5. Upload approved source images into the model Drive folder when raw Drive upload is available.
6. Index every image in Git with filename, role, approval status, checksum, and Drive file ID.
7. Only then request HeyGen avatar creation approval.
